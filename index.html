What is Jonah?

Jonah is a discord bot that allows users to talk to Jonah and get a responce from an AI simply by mentioning it in a server or talking to it in a DM.

What can Jonah do?

Jonah can talk with you like a real person would and responding to whatever you say to it. It also has 5 commands which users can use in a server or direct message:

  • /information: gives details about the bot
  • /me: gives details about the you such as when you created your profile.
  • /nuke [optional: amount] (admin only): allows admins to remove all the comments or a certain about of comments from a channel in a server.
  • /dm starts a DM with the user!
  • /joke tells you a random joke!
